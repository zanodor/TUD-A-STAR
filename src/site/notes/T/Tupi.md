# Tupi

A tupi indiánok neve Tu-Pi Nap-Csillag értelmű. Tub-I olvasata is lehet, ahogy [[T/TUB\|TUB]]/[[D/DUB\|DUB]] = tűz. Tupi, [[T/TUP\|TUP]]-i lehet tehát Tüzesek, Tüzecskék értelmű is.
