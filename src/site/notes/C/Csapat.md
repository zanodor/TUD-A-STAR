# Csapat

[[C/Csap\|Csap]]-At annyi lehetne, mint az egész Le-Csapott része.  
Egyúttal a harcos egységek lecsapnak az ellenségre.  
Hasonló hangzású a [[C/Csoport\|csoport]].
