# Corrie

Kör alakú vájat domb, hegy oldalában, skót angolságban (lásd még általánosabb [[C/Cirque\|cirque]]). Kör, kerek, karika szavaink láthatók benne. Lásd még (welsh-angol) [[C/Cwm\|cwm]].
